CRABY

A tiny pixel-art crab that watches your development workflow — a presence interface any local brain can use as its face. Claude Code is the flagship integration: see what it's up to and answer back without touching the terminal.

View on GitHub Leia em Português

Install

git clone https://github.com/duperez/crab-companion.git cd crab-companion ./install.sh

Requires macOS 13+, Xcode Command Line Tools, jq and Claude Code. ~300 KB, zero dependencies, MIT.

What Craby does

🖥️ Always in the corner of your eye

Floats above every app and every Space — even fullscreen. Never steals focus, no Dock icon.

💬 Answer permissions from a bubble

When Claude asks for permission, click Allow / Always allow / Deny right under Craby. Dangerous commands get a red border. No answer? Falls back to the terminal — Craby never decides alone.

❓ Questions too

Multiple-choice and free-text questions through the bubble, via a simple local HTTP API any script can use.

🗂️ Multi-session scoreboard

Several sessions at once? Craby shows the highest-priority state, dots for parallel work, and a tooltip per project. Click to jump to the session that needs you.

🐣 A brood of subagents

Spawn subagents and watch baby crabs hatch from eggs below Craby — they scuttle while the subagent runs, retire with a tiny cane and vanish in a star poof (gray poof if it failed). No other coding pet does this.

📍 Menu bar twin

A mini animated Craby in the menu bar, always in sync. Collapse the floating one when you need a clean screen.

🔔 Subtle sounds

A soft pling when work finishes, a ping when Claude needs you. Toggle anytime from the menu bar.

🧩 Addons

Drop a folder with an addon.json and an executable and Craby learns new tricks. github-watch and live-watch ship out of the box — or build your own in the visual builder at the playground (localhost:4923).

🔍 Keeping watch

Put Craby on vigil with /watch: he raises a little magnifying glass while observing, and sits next to a crate when he's watching Docker containers.

📜 Open protocol

Everything runs over a documented local HTTP protocol (PROTOCOL.md). Any script, tool or local brain can drive Craby — Claude Code is just the first face-wearer.

📊 Daily stats & plan usage

Tasks finished, projects, time worked, recent events — plus how much of your Claude plan you've used today.

📱 Phone alerts when away

Optional ntfy integration: if you're not at the Mac and Claude needs you, your phone gets a ping.

🎨 Custom sprite packs

Sprites are text grids. Drop a sprites.json to reskin Craby entirely — cat, octopus, you name it.

💤 He's alive

Idle quirks, a sweat drop on long tasks, naps when nothing happens, a confetti party every time you git push — and when you swipe to another Space, he materializes out of a little magic cloud, poof.

How it works

Claude Code ── hooks ──> localhost:4923 ──> 🦀 Craby (floating + menu bar)
     ^                                            │
     └────────── decision (long-poll HTTP) ──────┘

UserPromptSubmit → typing on the laptop     Stop → celebration
Notification → waving for attention         PermissionRequest → bubble, waits for your click

The same open HTTP protocol (PROTOCOL.md) works for any brain, not just Claude Code —
and addons (a folder + addon.json + an executable) plug extra behaviors into Craby.
