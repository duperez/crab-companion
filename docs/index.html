CRABY

A tiny pixel-art crab that lives on top of your Mac screen and tells you what Claude Code is up to — and lets you answer back without touching the terminal.

View on GitHub Leia em Português

Install

git clone https://github.com/duperez/crab-companion.git cd crab-companion ./install.sh

Requires macOS 13+, Xcode Command Line Tools, jq and Claude Code. ~300 KB, zero dependencies, MIT.

What Craby does

🖥️ Always in the corner of your eye

Floats above every app and every Space — even fullscreen. Never steals focus, no Dock icon.

💬 Answer permissions from a bubble

When Claude asks for permission, click Allow / Deny right under Craby. Dangerous commands get a red border. No answer? Falls back to the terminal — Craby never decides alone.

❓ Questions too

Multiple-choice and free-text questions through the bubble, via a simple local HTTP API any script can use.

🗂️ Multi-session scoreboard

Several sessions at once? Craby shows the highest-priority state, dots for parallel work, and a tooltip per project. Click to jump to the session that needs you.

📍 Menu bar twin

A mini animated Craby in the menu bar, always in sync. Collapse the floating one when you need a clean screen.

🔔 Subtle sounds

A soft pling when work finishes, a ping when Claude needs you. Toggle anytime from the menu bar.

📊 Daily stats & levels

Tasks finished, projects, time worked, recent events — and Craby levels up from hatchling to legend as work piles up.

📱 Phone alerts when away

Optional ntfy integration: if you're not at the Mac and Claude needs you, your phone gets a ping.

🎨 Custom sprite packs

Sprites are text grids. Drop a sprites.json to reskin Craby entirely — cat, octopus, you name it.

💤 He's alive

Idle quirks, a sweat drop on long tasks, naps when nothing happens, and confetti when he levels up.
